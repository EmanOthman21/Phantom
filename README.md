## Installation
* ## Server
	* ####  Run
		```bash
		$ cd server/phantom
		
		# install
		$ npm install
		
		# development
		$ npm run start

		# watch mode
		$ npm run start:dev

		# production mode
		$ npm run start:prod

		```
